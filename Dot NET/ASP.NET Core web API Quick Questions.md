## 2. What is JWT Token?

**JWT (JSON Web Token)** is a **self-contained token** used for **stateless authentication** between client and server.

## 3. What is CORS?

**CORS (Cross-Origin Resource Sharing)** allows or restricts **cross-origin requests** between frontend and backend domains.
